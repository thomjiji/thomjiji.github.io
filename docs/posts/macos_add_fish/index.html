macOS add fish to $SHELL and set fish as default shell

2023-01-26 | Suggest Changes

Solution#

Add fish to $SHELL and set as default#

  1. Add the shell to /etc/shells with:
$ echo "/opt/homebrew/bin/fish" | sudo tee -a /etc/shells
  1. Change default shell with:
chsh -s /opt/homebrew/bin/fish

Modify $PATH with fish shell#

  1. 在 iTerm2 当中使用 fish shell 作为默认 shell，需要把 $PATH 加入到 ~/.config/fish/fish_variables
set -U fish_user_paths /usr/local/bin $fish_user_paths
  1. Type this command in terminal: fish_add_path /opt/homebrew/bin. It will add /opt/homebrew/bin to the ~/.config/fish/fish_variables.

Other useful commands#

which $SHELL

echo $PATH

Homebrew 下载的 Fish shell 的 path 在 /opt/homebrew/bin/fish 而不是像 fish documentation 说的那样在 /usr/local/bin/fish。
